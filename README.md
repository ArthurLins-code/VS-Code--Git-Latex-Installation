# VS Code, Git, and LaTeX Installation Guide

A step-by-step setup manual for Windows users who want to build a complete working environment for editing and running R, writing LaTeX documents, and using Git and GitHub — all from Visual Studio Code. The guide assumes you already have R and RStudio installed and walks you through everything else: Git for Windows, a GitHub account, GitHub Desktop, VS Code, MiKTeX, and the essential VS Code extensions. It is written for academic researchers with zero command-line experience.
